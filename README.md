# 📖 Project Description

The **AI-Integrated Smart Grievance Management System** is a modern web application that enables citizens to register complaints online and allows administrators to manage, monitor and resolve grievances efficiently.

Unlike traditional paper-based complaint systems, this project integrates **Artificial Intelligence (AI)** to automatically classify complaints, predict their priority level, generate AI scores and improve complaint handling efficiency.

The system is designed for educational purposes while demonstrating how AI and modern web technologies can improve e-Governance and digital public services.

# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/surajkumarpaswan62/smart-grievance-portal.git
```

Move into folder

```bash
cd smart-grievance-portal
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate

```bash
venv\Scripts\activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Project

```bash
python app.py
```
