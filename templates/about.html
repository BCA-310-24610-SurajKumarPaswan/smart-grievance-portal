About Smart Grievance Portal

Project Name: Smart Grievance Portal

Objective: This project provides an online platform where citizens can register complaints, upload evidence, track complaint status using a Tracking ID, receive admin replies, and download FIR reports. The system helps government authorities manage public grievances efficiently.

Technologies Used:

  • Python
  • Flask
  • SQLite
  • HTML
  • CSS
  • JavaScript
  • Chart.js

Key Features:

  • AI Priority Detection
  • Complaint Tracking ID
  • Evidence Upload
  • Admin Dashboard
  • Complaint Timeline
  • PDF FIR Generation
  • Analytics Dashboard
  • Search & Filters
